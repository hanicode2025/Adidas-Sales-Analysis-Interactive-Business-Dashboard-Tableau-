# 📊 Adidas Sales Analysis & Interactive Dashboard

## 📌 Overview
This project presents a comprehensive sales analysis of Adidas data using an interactive Tableau dashboard. It explores revenue trends, product performance, regional sales distribution, and profitability to uncover actionable business insights.

## 🛠️ Tools Used
- Tableau  
- Data Cleaning & Processing  
- Data Visualization  
- Business Analytics  

## 📂 Project Structure
- dashboard/ → Tableau workbook (.twbx)  
- reports/ → Exported PDF (optional)  
- screenshots/ → Dashboard preview images  

## 📊 Key Metrics
- Total Sales: $899.90M  
- Total Units Sold: 2,478,861  
- Total Operating Profit: $332.13M  
- Average Price per Unit: $45.22  
- Product Categories: 6  

## 📈 Key Insights
- West region leads in both sales and operating profit, indicating strong market dominance  
- In-store sales outperform online and outlet channels, contributing the highest revenue  
- Men's Street Footwear and Men's Athletic Footwear generate the highest sales volume  
- Retailers like West Gear and Foot Locker contribute significantly to overall revenue  
- Seasonal trends show fluctuating monthly sales, with peaks around mid-year  

## 📊 Dashboard Features
- Sales trends over time (monthly analysis)  
- Product-wise performance comparison  
- Regional and state-level sales insights  
- Sales method breakdown (Online, In-store, Outlet)  
- Operating profit analysis by region  
- Retailer contribution analysis
